IBM Articles

I wrote These articles upon joining the IBM Streaming Analytics team. Web links are available for those that are still online, whereas those that have been decommissioned have PDF links to the original articles.

Integrating IBM Streams with Cloudant and other RESTful Services

(Online, PDF)

This article instructs how to make GET and POST requests from IBM Streams to any other authenticated RESTful service.

Get Started with Streaming Analytics + Message Hub

(PDF)

These instructions show how to communicate from IBM Streaming Analytics service to other services via MessageHub, IBM's Kafka messaging service.

Access Messsage Connect data sources from Streaming Analytics

(PDF)

This article shows how to communicate from Streaming Analytics via Kafka using IBM Message Connect - slightly simpler although less powerful than the Message Hub article above.

Using Streaming Analytics with JDBC-enabled Bluemix databases

(PDF)

This article shows the reader how to persist and retrieve Streaming Analytics data using JDBC databases

Get started with Streaming Analytics and BigInsights on Bluemix using HDFS

(PDF)

BigInsights is IBM's Hadoop cloud offering. This article shows how to store and retrieve BigInsights data in Streaming Analytics app from BigInsights.

Streaming Analytics metrics using the Streams REST API

(PDF)

This article shows how to use the Streams REST API to get information about Streaming Analytics via the underlying Streams service API.
